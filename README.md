# Github Analyze LLM

본인의 깃허브 계정을 연결해 본인이 참여한 프로젝트에서 구현한 것들을 바탕으로 대화할 수 있는 LLM

## 🔨 주요 기능
- 본인이 참여한 Repository 불러오기
- 선택한 Repository를 기반으로 대화 기능


## ⚒️ 세부 기능
- 본인이 참여한 Repository 가져오기
- LLM에 사용할 Repository 설정
- 해당 Repository 내용을 통해 LLM 생성
- 대화를 통해 원하는 대답 생성

## 💻 기술 스택
- Python 3.11.9 (Jupytor)
- OpenAI_API
